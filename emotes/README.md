# Instruções para adicionar emotes

Esta pasta deve conter as imagens dos emotes do Clash Royale.

## Como adicionar emotes:

1. Encontre ou crie imagens de emotes do Clash Royale
2. Salve as imagens no formato PNG (preferencialmente com fundo transparente)
3. Renomeie cada imagem com o **mesmo nome do gesto** correspondente

## Exemplo:

Se você capturou os seguintes gestos:
- thumbs_up
- peace
- fist
- open_palm
- pointing
- ok_sign

Então você deve ter arquivos:
- thumbs_up.png
- peace.png
- fist.png
- open_palm.png
- pointing.png
- ok_sign.png

## Onde encontrar emotes:

- Pesquise por "Clash Royale emotes PNG" no Google
- Use sites de recursos de jogos
- Crie seus próprios emotes personalizados

## Formatos suportados:

- PNG (recomendado - suporta transparência)
- JPG
- JPEG

## Tamanho recomendado:

- 200x200 pixels ou maior
- O sistema redimensionará automaticamente se necessário
